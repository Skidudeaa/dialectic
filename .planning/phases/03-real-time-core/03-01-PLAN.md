<task type="auto">
  <name>Task 3: Implement presence and receipt handlers</name>
  <files>dialectic/transport/handlers.py</files>
  <action>
Add four new handler methods to the MessageHandler class:

1. **_handle_presence_heartbeat(self, conn, payload)**:
   - Update user_presence table: INSERT ON CONFLICT DO UPDATE
   - Set status = 'online', last_heartbeat = NOW()
   - Broadcast presence_update to room (exclude sender)
   - Payload: {user_id, status: 'online', timestamp}

2. **_handle_presence_update(self, conn, payload)**:
   - Extract status from payload ('online' | 'away' | 'offline')
   - Update user_presence table
   - Broadcast presence_update to room (exclude sender)
   - Payload: {user_id, status, timestamp}

3. **_handle_message_delivered(self, conn, payload)**:
   - Extract message_id from payload
   - Insert into message_receipts (ON CONFLICT DO NOTHING)
   - Fetch message sender from messages table
   - Send delivery_receipt to sender only (not broadcast)
   - Payload: {message_id, status: 'delivered', recipient_id}

4. **_handle_message_read(self, conn, payload)**:
   - Extract message_id from payload
   - Insert into message_receipts with receipt_type = 'read'
   - Fetch message sender
   - Send read_receipt to sender only
   - Payload: {message_id, reader_id}

Add routing in the main handle() method to dispatch these message types to the new handlers.

Import datetime if not already imported. Use conn.user_id and conn.room_id from the Connection object.
  </action>
