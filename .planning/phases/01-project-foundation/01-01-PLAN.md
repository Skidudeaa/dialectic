  <files>
    mobile/eslint.config.js
    mobile/.prettierrc
    mobile/package.json
  </files>
  <action>
    1. Install ESLint with Expo config (from /root/DwoodAmo/mobile):
       ```bash
       npx expo lint
       ```
       This auto-installs eslint-config-expo and creates initial config.

    2. Install Prettier and integration:
       ```bash
       npx expo install prettier eslint-config-prettier eslint-plugin-prettier --dev
       ```

    3. Create mobile/.prettierrc:
       ```json
       {
         "semi": true,
         "trailingComma": "es5",
         "singleQuote": true,
         "printWidth": 100,
         "tabWidth": 2,
         "useTabs": false,
         "arrowParens": "always",
         "endOfLine": "lf"
       }
       ```

    4. Update mobile/eslint.config.js to flat config format with Prettier:
       ```javascript
       const { defineConfig, globalIgnores } = require('eslint/config');
       const expoConfig = require('eslint-config-expo/flat');
       const eslintPluginPrettierRecommended = require('eslint-plugin-prettier/recommended');

       module.exports = defineConfig([
         globalIgnores(['dist/*', 'node_modules/*', '.expo/*', 'coverage/*']),
         expoConfig,
         eslintPluginPrettierRecommended,
       ]);
       ```

    5. Add lint script to package.json if not present:
       ```json
       "scripts": {
         "lint": "expo lint",
         "lint:fix": "expo lint --fix"
       }
       ```

    6. Run lint to verify:
       ```bash
       npx expo lint
       ```
       Fix any auto-fixable issues with `npx expo lint --fix`.
  </action>

<task type="auto">
  <name>Task 3: Configure Jest and Testing Library</name>
  <files>
    mobile/jest.config.js
    mobile/jest-setup.js
    mobile/__tests__/HomeScreen-test.tsx
    mobile/package.json
  </files>
  <action>
    1. Install Jest and Testing Library (from /root/DwoodAmo/mobile):
       ```bash
       npx expo install jest-expo jest @types/jest @testing-library/react-native --dev
       ```

    2. Create mobile/jest.config.js:
       ```javascript
       module.exports = {
         preset: 'jest-expo',
         setupFilesAfterEnv: ['./jest-setup.js'],
         testPathIgnorePatterns: ['/node_modules/', '/app/'],
         collectCoverageFrom: [
           '**/*.{ts,tsx}',
           '!**/node_modules/**',
           '!**/app/**',
           '!**/*.d.ts',
         ],
         transformIgnorePatterns: [
           'node_modules/(?!((jest-)?react-native|@react-native(-community)?)|expo(nent)?|@expo(nent)?/.*|@expo-google-fonts/.*|react-navigation|@react-navigation/.*|@unimodules/.*|unimodules|sentry-expo|native-base|react-native-svg)',
         ],
       };
       ```

    3. Create mobile/jest-setup.js:
       ```javascript
       import '@testing-library/react-native/extend-expect';
       ```

    4. Create mobile/__tests__/HomeScreen-test.tsx:
       ```typescript
       import { render, screen } from '@testing-library/react-native';
       import Index from '../app/(tabs)/index';

       describe('<Index />', () => {
         it('renders chat tab content', () => {
           render(<Index />);
           // Update this assertion based on actual content
           expect(screen.getByText(/chat/i)).toBeTruthy();
         });
       });
       ```

    5. Add test scripts to package.json:
       ```json
       "scripts": {
         "test": "jest",
         "test:watch": "jest --watchAll",
         "test:coverage": "jest --coverage"
       }
       ```

    6. Run tests to verify setup:
       ```bash
       npm test
       ```

    IMPORTANT: Test files go in __tests__/, NOT in app/. Files in app/ become routes.
  </action>
  <verify>
    From /root/DwoodAmo/mobile:
    - `npm test` runs and passes (at least 1 test)
    - jest.config.js exists with jest-expo preset
    - __tests__/HomeScreen-test.tsx exists
  </verify>
  <done>
    Jest and Testing Library configured. Tests run and pass.
