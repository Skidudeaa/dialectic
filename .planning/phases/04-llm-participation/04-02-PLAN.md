<task type="auto">
  <name>Task 4: Wire WebSocket to dispatch LLM events</name>
  <files>mobile/services/websocket/index.ts</files>
  <action>
Update WebSocketService to dispatch LLM events to registered callbacks.

1. Add private field for LLM callback:
   ```typescript
   private llmEventCallback: ((type: string, payload: Record<string, unknown>) => void) | null = null;
   ```

2. Add method to register callback:
   ```typescript
   onLLMEvent(callback: (type: string, payload: Record<string, unknown>) => void): void {
     this.llmEventCallback = callback;
   }
   ```

3. Add method to unregister:
   ```typescript
   offLLMEvent(): void {
     this.llmEventCallback = null;
   }
   ```

4. In handleMessage, after parsing, check for LLM event types and dispatch:
   ```typescript
   // Dispatch LLM events to registered callback
   const llmEventTypes = ['llm_thinking', 'llm_streaming', 'llm_done', 'llm_error'];
   if (llmEventTypes.includes(message.type) && this.llmEventCallback) {
     this.llmEventCallback(message.type, message.payload);
   }
   ```

This allows the chat screen to register handlers from useLLM when mounting.
