<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
Complete desktop infrastructure:
- Monorepo with packages/app (shared), packages/mobile, packages/macos, packages/windows
- Platform service abstractions (secure storage, database, notifications)
- macOS implementation with Keychain, SQLite, menu bar
- Windows implementation with encrypted MMKV, SQLite, WinRT notifications
- Desktop UX components (keyboard shortcuts, hover actions, context menu, drag-drop, sidebar)
- Desktop layout with centered content and window persistence
  </what-built>
  <how-to-verify>
**On macOS:**
1. Open terminal at project root
2. Run `cd packages/macos && yarn macos` (or open macos/Dialectic.xcworkspace in Xcode)
3. App should launch with window showing "Dialectic for macOS"
4. Test keyboard shortcuts: Cmd+N, Cmd+F should log to console
5. Test sidebar collapse by clicking toggle
6. Drag a file onto the window - should log dropped files

**On Windows:**
1. Open terminal at project root
2. Run `cd packages/windows && yarn windows` (or open windows/Dialectic.sln in Visual Studio)
3. App should launch with window showing "Dialectic for Windows"
4. Test keyboard shortcuts: Ctrl+N, Ctrl+F should log to console
5. Test sidebar collapse by clicking toggle
6. Drag a file onto the window - should log dropped files

**Expected results:**
- Windows render correctly with native title bar
- Sidebar animates when toggling
- Keyboard shortcuts trigger (check console/logs)
- Drop zone highlights when dragging files over

Note: If not on macOS/Windows, verify configurations are correct and note that actual build requires platform machine.
  </how-to-verify>
